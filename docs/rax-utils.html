rax-utils CLI¶

usage: rax-utils [-h] [-f] [-s] [-d] [-l] [-n [NODES [NODES ...]]] [-v]
                 [--version]
                 env

Positional Arguments¶

env

Environment identifier

Named Arguments¶

-f, --force

Use this flag to force the action to occur

Default: False

-s, --stop

Stop servers (requires use of -f)

Default: False

-d, --destroy

Destroy servers (requires use of -f)

Default: False

-l, --list

List servers

Default: False

-n, --nodes

List of nodes (servers) to perform actions on

-v, --verbose

Verbosity (-v, -vv, etc)

Default: 0

--version

show program’s version number and exit
