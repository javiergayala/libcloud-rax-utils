class RaxConfigParser.LoginInfo(identity_type: str, username: str, api_key: str = None, region: str = None)¶

Class for keeping track of user login info.

Parameters

  • identity_type (str) – Identity Type (should be “rackspace_cloud”)

  • username (str) – Username for connecting to RAX Public Cloud

  • api_key (str) – API Key to authenticate with

  • region (str) – Region to connect to
